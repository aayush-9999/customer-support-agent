# Returns & Refunds — Facts

## Refund Timelines (clock starts when Leafy receives the item)

| Method | Timeline |
|---|---|
| Original payment method | 5–7 business days after item received |
| Store credit | Instant on return approval + 10% bonus |
| Bank transfer | 7–10 business days after item received |

## Return Eligibility Rules
- Items must be unworn, unwashed, unaltered, in original packaging with all tags attached
- Non-returnable: underwear, swimwear, pierced jewellery, Final Sale items, gift cards
- Partial returns allowed — each item assessed independently
- Return window: 30 days (Bronze/Silver/Gold), 45 days (Platinum) from delivery date
- Items must be shipped back within 7 days of receiving the RMA

## Who Pays Return Shipping
- Defective / wrong item / not as described → Leafy provides free return label
- Changed mind / size issue → Customer pays
- Platinum tier → Leafy covers all returns regardless of reason

## Return Statuses
Requested → Approved (RMA issued) → Item Received → Refund Issued | Rejected

## Rejection Reasons
Outside return window | Customer-caused damage | Tags removed or packaging missing | Non-returnable category

## Edge Cases
- Store credit can be chosen any time before refund is issued (not after)
- No direct exchanges — return + reorder
- Defective items: report within 72h of delivery for priority. Usually no need to return — agent discretion
- Customer didn't use a Leafy return label but should have (defective/wrong item): escalate to review reimbursement
- Loyalty points: awarded only after return window closes. Points for returned items are not awarded.
