# Return & Refund Policy — Rules

## Eligibility
- Return window: 30 days from delivery (Platinum: 45 days)
- Items must be: unworn, unwashed, unaltered, original packaging, all tags attached
- Non-returnable: underwear, swimwear, pierced jewellery, Final Sale items, digital gift cards, customer-damaged items

## Refund Methods & Timelines

| Method | Timeline | Notes |
|---|---|---|
| Original payment method | 5–7 business days after item received | Minus original shipping for change-of-mind |
| Store credit | Instant on approval | +10% bonus on item value |
| Bank transfer | 7–10 business days after item received | Requires verified bank details |

## Return Shipping
| Reason | Covered By |
|---|---|
| Defective / damaged / wrong item / not as described | Leafy (free label) |
| Changed mind / size issue | Customer pays |
| Platinum tier (any reason) | Leafy (free label) |

## Tier Return Benefits

| Tier | Window | Priority Processing | Free Label |
|---|---|---|---|
| Bronze / Silver | 30 days | No | Defective only |
| Gold | 30 days | Yes | Defective only |
| Platinum | 45 days | Yes | All returns |

## Rejection Reasons
Outside window | Customer-caused damage | Tags removed or packaging missing | Non-returnable category

## Rules
- Items must be shipped back within 7 days of RMA
- No direct exchanges — return + reorder
- Defective items: report within 72h of delivery. Return usually not required.
- Store credit option available any time before refund is issued
