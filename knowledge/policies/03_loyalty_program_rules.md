# Leafy Rewards — Rules

## Earning Points

| Action | Points |
|---|---|
| Every $1 spent | Bronze: 10 / Silver: 12 / Gold: 15 / Platinum: 20 |
| Verified product review | 50 |
| Referring a friend (on their first purchase) | 200 |
| Birthday bonus (once/year) | 100 |
| Account creation (one-time) | 50 |

Points credited after delivery AND return window closes. Returned items earn no points.

## Tier Thresholds (based on lifetime points earned — spending points doesn't reduce tier progress)

| Tier | Threshold |
|---|---|
| Bronze | 0–249 |
| Silver | 250–749 |
| Gold | 750–1,499 |
| Platinum | 1,500+ |

Tier maintained for 12 months from the date it was reached, even with reduced activity.

## Tier Benefits Summary

| Benefit | Bronze | Silver | Gold | Platinum |
|---|---|---|---|---|
| Return window | 30 days | 30 days | 30 days | 45 days |
| Free return labels | Defective only | Defective only | Defective only | All returns |
| Free shipping | $75+ threshold | $75+ threshold | All standard | All express |
| Support priority | Standard | Standard | Priority queue | Top of queue |
| Early sale access | Member events | 24h before public | 48h before public | 72h before public |
| Birthday bonus | No | Yes (100 pts) | Yes (100 pts) | Yes (100 pts) |

## Redeeming Points

| Points | Discount |
|---|---|
| 500 | $5 |
| 1,000 | $10 |
| 2,000 | $22 (+10% bonus) |
| 5,000 | $60 (+20% bonus) |

- Minimum $20 order value to redeem
- Cannot combine with promo codes (unless campaign marked "combinable")
- Redeemed points non-refundable if order is returned

## Expiry & Disputes
- Points expire after 18 months with no purchase activity
- Warning email sent 60 days before expiry. Expired points cannot be reinstated.
- Disputes: contact within 30 days with order ID. Agent can view loyalty ledger and credit manually if warranted.
